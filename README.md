# sbom2vans

## Build

```
$ go build -o sbom2vans ./cmd/sbom2vans
```


## Usage

```
$ ./sbom2vans --name Alice
Hello, Alice!



$ ./sbom2vans -h
A simple CLI tool

Usage:
  sbom2vans [flags]

Flags:
  -h, --help          help for sbom2vans
  -n, --name string   Specify a name (default "World")
```
